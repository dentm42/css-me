Add custom CSS to any single page or post

== Description ==
A simple Wordpress plugin that allows users to add custom css to any single page or post

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload entire `css-me` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's it, you'll see a Add CSS metabox on all add or edit posts/pages.
